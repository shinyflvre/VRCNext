## What's New in 2026.1.4

### Avatars Tab — Rose Database
- New **Rose Database** tab between Favorites and Search, powered by the community avatar list
- Fetches avatars on demand with a loading state; cached until manually refreshed via the Refresh button
- Live search across avatar name, author, and tags
- Tag badges with distinct colors per category: **Fish** (white), **Rose** (red), **Arcade** (pink→cyan gradient), **Vapor** (cyan), **Cheese** (yellow), **Cosmic** (blue)
- Tags always displayed in a consistent order: Fish → Rose → Arcade → Vapor → Cheese → Cosmic
- Favorite/Unfavorite works the same as all other avatar tabs
